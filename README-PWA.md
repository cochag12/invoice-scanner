## 🖼️ יצירת אייקונים

1. גש ל: https://www.favicon-generator.org/
2. העלה תמונה או לוגו
3. הורד את icon-192.png ו-icon-512.png
4. העלה ל-GitHub

1. גש ל: https://favicon.io/emoji-favicons/camera/
2. בחר 📸 (מצלמה)
3. הורד
4. שנה שם ל: icon-192.png ו-icon-512.png

### אפשרות 3: בלי אייקונים (זמני)

האפליקציה תעבוד גם בלי! פשוט תראה אייקון ברירת מחדל.

## 🚀 איך להתקין במכשיר:

### Android (Chrome):
1. פתח: https://cochag12.github.io/invoice-scanner/
2. תפריט (⋮) → **"התקן אפליקציה"** או **"הוסף למסך הבית"**
3. לחץ **"התקן"**
4. ✅ האייקון יופיע במסך הבית!

### iPhone (Safari):
1. פתח: https://cochag12.github.io/invoice-scanner/
2. כפתור שיתוף 📤
3. גלול ל-**"הוסף למסך הבית"**
4. **"הוסף"**
5. ✅ האייקון יופיע!

## 🐛 פתרון בעיות:

### "לא מציע להתקין"
- ודא ש-manifest.json קיים
- ודא ש-sw.js קיים
- רענן את הדף (Ctrl+Shift+R)

### "אייקון לא מופיע"
- ודא שהאייקונים הועלו
- שנה את הנתיב ב-manifest.json אם צריך
- רענן ונסה שוב

### "לא עובד אופליין"
- פתח את האפליקציה עם אינטרנט פעם אחת
- Service Worker צריך להתקבל
- אז זה יעבוד אופליין
